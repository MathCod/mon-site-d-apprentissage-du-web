Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le web, se traduisant par "toile" en français, apparaît en 1991 grace à son créateur Tim Berners-Lee. Le web représente l'ensemble des données reliées par des liens hypertextes sur Internet. Souvent comparée à une toile d'araignée symbolisant toutes les intersections et chemins possibles, d'où l'expression "surfer sur la toile".

Internet

Internet représente le réseaux qui permet à des ordinateurs et à des serveurs de communiquer efficacement ensemble au moyen d'un protocole de communication commun : "l'adresse IP". Il contient plusieurs services comme le web, les e-mails, le FTP et tant d'autres.

Cloud

Signifiant "Nuage" en français, le cloud conserve vos données et services, et est accessible de partout et à tout moment avec une connexion internet. il peut stocker en réseau (l'inverse du local) des photos (Dropbox, Google Drive), des services d'e-mail (Gmail, Outlook) par erxemple.

W3C

Créé en 1994, le W3C (ou World Wide Web Consortium) est un organisme international destiné à gérer les évolutions du web et de définir ses standards techniques. Surtout, ils déterminent les mêmes règles pour tous les développeurs du monde.

Partie 2 - Distinguez les langages du Web

Langages client

Les langages client servent à fournir des informations de base pour la construction des sites web. C'est un peu comme des plans d'architecte. il est exécuté côté client (sur l'ordinateur de l'utilisateur). Ces langages sont principalement : HTML, CSS et JavaScript.

Navigateurs

Un navigateur est un logiciel permettant d'accéder à des informations sur le Web. Lorsqu'un utilisateur demande des informations, le navigateur Web récupère les données d'un serveur, puis affiche la page Web sur son écran. On peut citer par exemple : Chrome, Safari, Edge, Firefox ou encore opera.

Langages serveur

Un langage serveur est un type de langage de programmation qui est exécuté côté serveur (ordinateur qui héberge un site). Les langages serveur les plus courants : PHP, Python, Ruby, Java ou encore Node.js.

Frameworks

On peut voir les Frameworks comme des bibliothèques et des outils. Ils permettent de structurer et de développer des applications logicielles. Ils fournissent une base pour le développement, offrant des solutions préconçues aux problèmes courants rencontrés lors de la création de logiciels. On peut citer par exemple : js, React, Angular, Django ou Ruby on Rails.

Bases de données

Une base de données est un ensemble d'informations qui est organisé de manière à être facilement accessible, géré et mis à jour. Elle est utilisée par les organisations comme méthode de stockage, de gestion et de récupération d'informations.

Site responsive

Un site responsive est un site web dont la conception vise, grâce à différents principes et techniques, à offrir une consultation confortable sur des écrans de tailles très différentes (par exemple sur ordinateur, tablette et aussi téléphone).

Application mobile

Une application est une alternative au site web responsive. elle est plus performante sur l'appareil quelle vise (téléphone ou tablette par exemple) mais implique le développement d'un nouveau logiciel en plus d'un site.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Un serveur est un ordinateur qui met des ressources, des données, des services ou des logiciels à la disposition d'autres ordinateurs, qualifiés de "clients", sur un réseau. Il y en a partout dans le monde et sont en général très sécurisés dans des datacenters. Emboîtés les uns sur les autres dans des compartiments appelés "baies", ils Ne disposent en général pas d'écrans.

Adresse IP

Une adresse IP est un numéro d'identification unique attribué à chaque périphérique connecté à un réseau informatique. L'adresse IP est à l'origine du système d'acheminement des paquets de données sur Internet.

Nom d’hôte et DNS

Un nom d'hôte est une étiquette unique attribuée à un appareil connecté à un réseau informatique. Il sert d'identificateur plus simple à lire qu'une adresse IP, ce qui vous permet de le distinguer facilement des autres appareils du réseau. Le DNS permet de traduire le nom d’hôte en adresse IP.

Protocoles

Un protocole est un ensemble de règles qui régissent les échanges de données ou le comportement collectif de processus ou d'ordinateurs en réseaux. Il a pour but de réaliser une ou plusieurs tâches pour un fonctionnement harmonieux.

Partie 4 - Identifiez les métiers du web

Développeurs

Un développeur est un informaticien qui réalise des logiciels ou sites web à l'aide de langages de programmation. Il travaille à partir d'un cahier des charges.
spécialités

Product Managers et chefs de projets

Un Product manager s'assure de la coordination des projets en flux tendus, et doit également traiter les retours des clients et est en lien avec les développeurs et l'équipe marketing pour les différents besoins du projet.
Le Chef de projet est responsable de l'élaboration du plan de projet, incluant la définition des objectifs, la portée, les échéanciers, et les budgets. Il doit identifier les ressources nécessaires, établir une équipe de projet, et définir les rôles et responsabilités de chaque membre.

Designers

Les designers font en sorte que le site soit le plus simple d’utilisation possible, ce sont les professionnels du design et de l’ergonomie des pages web ou des applications.

Testeurs

Le testeur est chargé de concevoir, d'exécuter et d'automatiser les tests pour garantir la qualité des logiciels ou des produits. Il identifie les bugs et les problèmes de performance afin de les résoudre avant la mise en réseau du logiciel.

Techniciens informatique

Le technicien informatique a un rôle polyvalent. Il s'occupe de la gestion du parc informatique, du support technique et de l'accès aux services réseaux des entreprises.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

Il permet de définir le visuel du site, en modifiant les couleurs, la police, les tailles, la dosposition, etc.

JavaScript

Front-End

Il permet d’ajouter de l’interaction sur le site comme des menus déroulants, affichages de messages lors de la validation de formulaires, etc.

JavaScript

Back-End

Il gère la logique côté serveur d'une application, y compris les interactions avec la base de données. Son utilisation en Back-End est possible grâce à Node.js

PHP

Back-End

Côté du serveur, son rôle est de générer une page HTML et CSS, qui sera ensuite envoyée au client. On peut installer des Frameworks comme Symfony, Zend...

Java

Back-End

Language différent de PHP, mais ayant la même fonction que ce dernier. Il possède différents Frameworks comme Java EE (ou J2EE).

Python

Back-End

Côté du serveur, il est réputé pour sa simplicité et sa lisibilité. Il possède le Frameworks Django.

C#

Back-End

Langage de programmation orienté multiplateforme moderne, novateur et open-source. Il possède le Frameworks ASP .NET.

SQL

Bases de données

Permet de "discuter" ou faire des requetes à une base de données. Il existe plusieurs logiciels comme MySQL, Oracle, SQLite, PostgreSQL, etc.

Boîte à outils du développeur

  • IDE comme VS Code : Un IDE, ou "integrated development environment" (environnement de développement intégré), est comme un éditeur de texte (comme Notepad++) mais en plus poussé et avec plus de fonctionnalités pour écrire du code.
  • Github : GitHub est une plateforme open source de gestion de contrôle de versions et de collaboration destinée aux développeurs de logiciels. Il permet de stocker ses travaux, avoir un suivi et partager à ses collaborateurs.
  • Librairies / bibliothèques : Dans le domaine de la programmation, une librairie désigne un entrepôt de code pré-construit utilisable par un public de programmeurs. Ces librairies ont pour but de faciliter le processus de programmation et d'accélérer le développement.
  • Frameworks : Ce sont des outils. Ils permettent de structurer et de développer des applications logicielles. Ils fournissent une base pour le développement, offrant des solutions préconçues.
    Il existe Django pour Python, Symfony pour PHP ou encore Java EE pour Java.
